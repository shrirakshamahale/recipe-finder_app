# recipe-finder_app

A React + Vite powered app that allows users to browse recipes by categories, explore detailed instructions, and discover new dishes through a clean and user-friendly interface.

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Scripts

- `npm install` – install dependencies  
- `npm run dev` – run the development server  
- `npm run build` – build for production

## Development

This project was bootstrapped with the React + Vite template.
